# Infrastructure Documentation

This module serves as a placeholder for infrastructure documentation or automated documentation generation tools (like `terraform-docs`).

## Current Architecture

*   **VPC**: Network foundation.
*   **IAM**: Security and access control.
*   **S3**: Object storage (Assets, Frontend).
*   **RDS**: Relational Database (PostgreSQL).
*   **ElastiCache**: In-memory caching (Redis).
*   **ECS/ECR**: Container orchestration and registry.
*   **ALB/ACM/Route53**: Load balancing, DNS, and SSL.
